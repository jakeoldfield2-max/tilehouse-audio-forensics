# Tilehouse Audio Forensics

First-draft single-page site for **Tilehouse Audio Forensics** — independent expert-witness audio analysis & forensic linguistics for UK criminal, civil, and family law proceedings. A trading style of Tilehouse Audio Ltd, visually linked to [Tilehouse Studios](https://www.tilehousestudios.com).

This is an early draft for review. The whole site is a single self-contained file, `index.html`.

## View it locally

Open `index.html` in any browser, or serve it:

```sh
python -m http.server 8765
```

then visit <http://localhost:8765/index.html>.

## Notes

- Fonts (Fraunces / Inter) are temporary stand-ins for the licensed brand typefaces.
- The instruction form is a visual mockup and is not yet wired to a backend.
